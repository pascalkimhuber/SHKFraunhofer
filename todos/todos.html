2.2 Funktion von saveValues() in SolParallelData

The saveValues() method is called by 

ParameterParser::parseFile().

The parseFile method passes as arguments a keyword list and an identifier list. Example of the succession of these lists for the N2 longrange solver:

  1. keyword: coulomb, n2 identifier: state
  2. keyword: coulomb, n2 identifier: rcut
  3. keyword: coulomb, n2 identifier: idegree

Thus for the ScaFaCoS methods the succession of lists should be like:

  1. keyword: coulomb, fcs identifier: state
  2. keyword: coulomb, fcs identifier: method
  3. keyword: coulomb, fcs identifier: alpha

…
